#标题
Radeon R300 显卡开启 KMS 后播放视频触发 GPU 锁死

#内核版本
2.6.38

#问题现象
- iBook G4 (R300 显卡) 启用 KMS 后，使用 mplayer 播放视频时系统发生 GPU lockup，网络仍可访问，dmesg 显示 Oops/stack traces。
- 内核 2.6.39-rc7 下出现颜色异常（红变蓝），且硬件加速初始化失败。
- 使用参数 `radeon.dynclks=1` 时，分辨率被错误限制为 800x600。

#问题根因
未明确。开发者分析认为可能与 TTM (Translation Table Maps) 的 DMA API 使用问题或 PowerPC 架构下的字节序问题有关；2.6.39-rc7 中出现的颜色异常被确认为 X 驱动的字节序 bug；GPU lockup 具体原因在报告中未最终定位。

#解决方案
暂无正式修复。可尝试通过内核参数 `radeon.agpmode=-1` 或 `radeon.no_wb=1` 进行规避；开发者提供了测试补丁用于强制开启所有 GPU 时钟以排查动态时钟问题。
