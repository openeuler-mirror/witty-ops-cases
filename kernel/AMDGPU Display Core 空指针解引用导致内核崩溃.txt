#标题
AMDGPU Display Core 空指针解引用导致内核崩溃

#内核版本
4.15.10

#问题现象
系统在运行 Firefox、MPV 以及 Plasma Shell (KWin/X11，重度 OpenGL 负载) 时触发内核崩溃。日志显示 "BUG: unable to handle kernel NULL pointer dereference at 0000000000000000"。

#问题根因
故障发生在 AMDGPU Display Core (DC) 模块中。用户推测可能与 AMD Ryzen 1600X 处理器的 "Performance Marginality Problem" 硬件缺陷有关，此前曾出现过随机 MCE 和死机；AMD 开发者认为可能是 DC 驱动代码本身的缺陷。具体导致空指针的代码路径未明确。

#解决方案
暂无
