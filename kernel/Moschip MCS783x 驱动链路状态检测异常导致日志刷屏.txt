#标题
Moschip MCS783x 驱动链路状态检测异常导致日志刷屏

#内核版本
3.10.3-300.fc19.x86_64

#问题现象
- Moschip MCS7830 适配器在使用 NetworkManager 时出现异常行为
- 系统日志被大量 "carrier now OFF" 和 "carrier now ON" 消息填满
- 链路状态每 2 秒左右频繁切换一次

#问题根因
mcs7830 驱动中引入链路状态检测功能的两个提交存在缺陷：
- b1ff4f96fd1c63890d78d8939c6e0f2b44ce3113 (mcs7830: Implement link state detection)
- dabdaf0caa3af520dbc1df87b2fb4e77224037bd (mcs7830: Fix link state detection)
这些提交导致驱动错误地频繁上报链路状态变化。

#解决方案
回退上述两个提交的代码更改可规避该问题。截至报告时，Bug 状态为 NEW，暂无官方修复补丁。
