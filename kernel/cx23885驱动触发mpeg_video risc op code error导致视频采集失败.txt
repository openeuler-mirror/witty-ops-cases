#标题
cx23885驱动触发mpeg/video risc op code error导致视频采集失败

#内核版本
5.15.5（从5.13.7升级后出现）

#问题现象
- 升级内核后，使用cx23885模块的视频采集卡（如Hauppauge WinTV-quadHD、AVerMedia CE310B）无法正常工作。
- 应用程序（w_scan, mythtv, mplayer, ffmpeg）无法找到信号或录制过程中卡死。
- 系统日志报错："cx23885: cx23885[x]: mpeg risc op code error" 或 "video risc op code error"，并伴随DMA通道状态转储信息（如 cmds: init risc lo, risc pc lo 等）。

#问题根因
问题在内核版本 5.13.7 至 5.15.5 之间引入，具体触发代码变更未明确。错误日志显示RISC处理器操作码错误及DMA相关状态异常，推测与DMA处理逻辑变更有关。

#解决方案
临时规避方案：加载cx23885模块时添加参数 `dma_reset_workaround=2`。官方修复暂无。
