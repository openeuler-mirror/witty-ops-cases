#标题
AMD Raven IGPU 启用 stutter mode 导致系统冻结

#内核版本
5.2.0 至 5.14.0

#问题现象
- 系统在启动后 5 分钟内发生完全冻结，屏幕出现蓝色小点
- 必须强制重启才能恢复
- 受影响硬件为 AMD Ryzen 3 2200G (Raven) 集成显卡

#问题根因
Git bisect 定位到 commit 005440066f929ba0dca8f4e0aebfbf8daac592cc。该提交修改了 `amdgpu_pp_feature_mask`，同时启用了 gfxoff 和 stutter mode。经用户排查验证，问题并非由 gfxoff 引起，而是启用 stutter mode (bit 17) 导致特定 Raven 系列硬件出现冻结。

#解决方案
开发者提供了修复补丁（见 Bugzilla 附件 299287），用户测试确认该补丁有效。临时规避方案为修改内核参数，将 `amdgpu_pp_feature_mask` 设置为 `0xfffdbfff` 以禁用 stutter mode。
