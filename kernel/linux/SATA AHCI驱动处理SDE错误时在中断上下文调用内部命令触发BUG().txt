#标题
SATA AHCI驱动处理SDE错误时在中断上下文调用内部命令触发BUG()

#内核版本
3.0.8

#问题现象
在 SATA 3.0 AHCI 驱动下，连接 Port Multiplier 并启用 FBS 时，若设备报告 "Single Device Error (SDE)"，内核触发 BUG()。
Call Trace:
ahci_interrupt -> ahci_port_intr -> ahci_error_intr -> ata_link_online -> ata_phys_link_online -> sata_scr_read -> sata_pmp_scr_read -> sata_pmp_read -> ata_exec_internal -> ata_exec_internal_sg

#问题根因
`ahci_error_intr` 错误地在中断上下文中调用了 `ata_link_online`，导致最终调用 `ata_exec_internal_sg`。该函数尝试获取 tag 31 (ATA_TAG_INTERNAL) 时触发 `BUG()`，因为 `ata_exec_internal_sg` 不应在中断上下文中被调用。

#解决方案
暂无
