#标题
TCP连接建立时tcp_transmit_skb函数触发kernel BUG

#内核版本
3.16.7-ckt9 (Debian testing)

#问题现象
- 系统日志报告 "kernel BUG at net/ipv4/tcp_output.c:878!"
- 触发 "invalid opcode: 0000 [#2]" 异常，导致进程终止
- 故障发生在 Chrome_IOThread 进程中
- 调用栈显示异常发生在 TCP 连接建立阶段（tcp_connect -> tcp_transmit_skb）

#问题根因
内核在 `tcp_transmit_skb` 函数（`tcp_output.c` 第 878 行）触发了 `BUG()` 宏（对应汇编指令 `0f 0b`）。故障现场 RIP 指向 `tcp_transmit_skb+0x6ff`。具体的逻辑触发条件（为何进入该 BUG 分支）在报告中未明确。

#解决方案
暂无
