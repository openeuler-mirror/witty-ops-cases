#标题
ath9k驱动在WEP加密连接时触发内核WARNING

#内核版本
4.0.3-1-MANJARO

#问题现象
- 使用 ath9k 网卡连接 WEP 加密网络时，dmesg 持续输出内核 WARNING 日志
- 堆栈信息显示警告位于 `net/mac80211/wep.c:102` 的 `ieee80211_wep_add_iv` 函数
- 网络连接看似正常，但执行 git clone 等操作时偶发失败，报错 "The remote end hung up unexpectedly"

#问题根因
在 WEP 连接场景下，`mac80211` 模块在发送处理流程中，`ieee80211_wep_add_iv` 函数触发了内核 WARNING。具体触发 WARNING 的代码逻辑原因未在报告中明确说明。

#解决方案
暂无
