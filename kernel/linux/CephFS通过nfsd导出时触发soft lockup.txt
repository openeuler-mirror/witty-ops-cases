#标题
CephFS通过nfsd导出时触发soft lockup

#内核版本
3.10.0-957.5.1.el7.x86_64

#问题现象
- 系统日志报错 "NMI watchdog: BUG: soft lockup - CPU#3 stuck for 23s!"
- 故障发生在 nfsd 进程中，导致 CPU 卡死。
- 调用栈显示卡死路径：nfsd -> fh_verify -> exportfs_decode_fh -> ceph_fh_to_dentry -> __fh_to_dentry -> d_obtain_alias。

#问题根因
用户通过 knfsd 将 CephFS 导出给客户端使用。维护者指出 "exporting kcephfs via knfsd is not a well-tested configuration"，即该组合属于未经充分测试的配置，可能导致内核 soft lockup。

#解决方案
暂无。维护者建议确认是否可在主线内核上复现该问题。
