#标题
Block Layer 禁用 runtime-pm 导致挂起或电池供电时系统死锁

#内核版本
4.12.9

#问题现象
- 笔记本电脑无法正常挂起或在使用电池供电时无法使用
- 块设备停止响应，dmesg 报告命令超时
- 系统发生死锁：X server 冻结，无法切换 tty
- 无法正确同步磁盘，即使使用 magic sysrq 键也无效

#问题根因
经 git bisect 定位，问题由提交 3c6a05484d0daca74f880d9559e28236330525cf ("block: disable runtime-pm for blk-mq") 引入。该提交禁用了 blk-mq 的 runtime-pm 功能，导致在挂起或电池供电等特定电源管理场景下，块设备出现异常并引发系统死锁。

#解决方案
暂无（Bug 状态为 NEW，尚未修复）
