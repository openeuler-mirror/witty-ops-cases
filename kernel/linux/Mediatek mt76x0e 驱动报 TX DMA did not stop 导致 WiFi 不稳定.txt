#标题
Mediatek mt76x0e 驱动报 TX DMA did not stop 导致 WiFi 不稳定

#内核版本
5.4.0 (Debian), 6.12.6-arch1-1 (Arch)

#问题现象
- 系统启动后短时间内 WiFi 功能失效，网络不可用
- 系统挂起/恢复后，WiFi 无法关联接入点（AP）
- 内核日志中出现 "mt76x0e 0000:04:00.0: TX DMA did not stop" 以及 "Warning: MAC RX did not stop!" 报错
- 重启系统后 WiFi 可暂时恢复正常

#问题根因
未明确。用户推测可能与 WiFi 芯片初始化不完全或内存状态有关，但缺乏开发者确认的具体技术根因。

#解决方案
暂无
