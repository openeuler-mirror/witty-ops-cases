#标题
rt5659驱动因ID寄存器校验失败导致声卡无法工作

#内核版本
5.15.80

#问题现象
- 设备声卡无法正常工作。
- 内核日志报错：`rt5659 i2c-10EC5659:01: Device with ID register 801efa10 is not rt5659`。
- 音频驱动探测失败：`sof-audio-acpi-intel-byt 808622A8:00: error: no matching ASoC machine driver found`，最终 `sof_probe_work` 返回错误码 -19 (ENODEV)。

#问题根因
rt5659 驱动在 I2C 探测阶段读取设备 ID 寄存器，获取到的值为 `801efa10`，该值与驱动预期的 rt5659 设备 ID 不匹配，导致驱动校验失败并中止加载。后续 sof-audio-acpi-intel-byt 驱动也因此找不到匹配的 ASoC 机器驱动而加载失败。

#解决方案
暂无
