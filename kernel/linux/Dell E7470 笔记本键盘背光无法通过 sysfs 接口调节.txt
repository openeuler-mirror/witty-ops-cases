#标题
Dell E7470 笔记本键盘背光无法通过 sysfs 接口调节

#内核版本
4.9.0（评论中提及 4.10.5-200 版本亦受影响）

#问题现象
- Dell E7470 及 Precision 7510 机型上，键盘背光物理按键调节正常，但通过 /sys/devices/platform/dell-laptop/leds/dell::kbd_backlight/brightness 文件写入值无效或结果错误（例如写入 1 读出 2，写入 0 读出 2）。
- dmesg 日志中每次写入 brightness 时均报错："dell_laptop: Setting old previous keyboard state failed"。
- dmesg 日志报错："leds dell::kbd_backlight: Setting an LED's brightness failed (-6)"。

#问题根因
未明确（问题发生在 dell_laptop 驱动模块处理键盘背光亮度设置时，具体代码逻辑错误未在报告中详述）。

#解决方案
该问题已在内核主分支修复，相关修复提交为：https://github.com/torvalds/linux/commit/9216e0dcb5533a999d544d0af8661118e0588e1d。补丁链接：https://patchwork.kernel.org/patch/9695271/。
