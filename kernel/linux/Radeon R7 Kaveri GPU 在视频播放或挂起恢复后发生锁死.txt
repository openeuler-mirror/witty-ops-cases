#标题
Radeon R7 Kaveri GPU 在视频播放或挂起恢复后发生锁死

#内核版本
5.19.16-100.fc35.x86_64

#问题现象
- 系统运行过程中出现随机 GPU lockup
- 内核日志显示 "ring stalled for more than 10xxx msec" 以及 "GPU lockup (current fence id ... last fence id ...)"
- 故障常在 Firefox 播放视频或系统执行挂起/恢复操作后触发
- 禁用视频加速功能后问题消失

#问题根因
未明确。经排查，问题非 Mesa 版本回归引起，亦非固件更新导致；故障与视频加速功能紧密相关，但具体的内核驱动代码缺陷尚未定位。

#解决方案
暂无官方修复。临时规避方案为禁用视频加速功能。
