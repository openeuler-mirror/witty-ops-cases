#标题
xhci_hcd 初始化时 Host halt 超时导致 USB 3.0 端口不可用

#内核版本
5.4.0-53-generic, 5.8.0-29-generic

#问题现象
- 系统升级后丢失所有 USB 3.0 端口功能
- 内核日志报错：xhci_hcd 0000:0b:00.0: Host halt failed, -110
- 驱动加载失败：xhci_hcd: probe of 0000:0b:00.0 failed with error -110

#问题根因
xHCI 控制器在驱动初始化过程中执行停止命令时未及时响应，导致超时（错误码 -110），进而导致控制器设置失败。

#解决方案
暂无（维护者建议向 Ubuntu 发行版提交 Bug 报告，上游状态为 NEW）
