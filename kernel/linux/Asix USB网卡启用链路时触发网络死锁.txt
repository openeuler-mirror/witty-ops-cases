#标题
Asix USB网卡启用链路时触发网络死锁

#内核版本
5.15.5 LTS

#问题现象
- 使用 Asix AX88772B USB 网卡时，执行 `ip link set dev eth1 up` 或 `ifconfig eth1 up` 命令后进程卡死，无法终止
- 网络相关程序（如 ip, ifconfig, avahi 等）随之冻结，系统需重启恢复
- 该问题为回归问题，在 5.10.83 LTS 内核版本下表现正常

#问题根因
未明确。根据用户反馈，推测可能为 Asix 驱动内部锁定状态不一致导致死锁。

#解决方案
暂无官方修复。可通过降级内核至 5.10.83 LTS 版本规避该问题。
