#标题
pktcdvd设备写入数据时触发kernel BUG导致无响应

#内核版本
4.15 (受影响版本包括 4.19-rc7, 5.3.0, 5.4.0)

#问题现象
- 使用 pktsetup 设置 pktcdvd 设备并挂载后，一旦有数据写入，系统触发 kernel BUG
- 日志显示 "invalid opcode 0000 kernel BUG in pkt_make_request"
- 设备随后失去响应
- 问题在 i386 和 amd64 架构上均可复现

#问题根因
触发位置位于 `pkt_make_request` 函数，错误类型为 invalid opcode 0000。由于维护者缺乏相应的硬件环境进行调试，具体的代码逻辑缺陷未明确。

#解决方案
维护者建议移除 pktcdvd 驱动，暂无修复补丁。
