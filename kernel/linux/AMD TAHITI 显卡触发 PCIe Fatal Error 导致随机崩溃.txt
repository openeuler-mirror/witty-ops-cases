#标题
AMD TAHITI 显卡触发 PCIe Fatal Error 导致随机崩溃

#内核版本
4.15.0-20-generic

#问题现象
- MacPro6,1 平台上使用 AMD TAHITI 显卡时系统随机崩溃
- 内核日志显示 PCIe 链路异常断开
- 报告 PCIe 总线致命错误：Surprise Down Error 和 Completion Timeout
- radeon 驱动报错 "atombios stuck in loop for more than 5secs aborting"
- AER 机制尝试恢复设备失败

#问题根因
系统日志显示 PCIe 总线发生 Transaction Layer 致命错误，具体表现为 Surprise Down Error 和 Completion Timeout。这导致 PCIe 链路断开，radeon 驱动在尝试执行 Atombios 指令时陷入死循环，最终导致设备恢复失败。具体触发条件未明确。

#解决方案
暂无
