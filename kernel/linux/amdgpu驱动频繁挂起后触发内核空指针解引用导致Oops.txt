#标题
amdgpu驱动频繁挂起后触发内核空指针解引用导致Oops

#内核版本
5.9.11

#问题现象
- RX 480 显卡每隔几小时发生随机 GPU 挂起，内核 5.9 版本后频率增加
- 系统日志显示 "[drm] Bailing on TDR for s_job..." 后紧接着报错 "BUG: kernel NULL pointer dereference, address: 0000000000000020"
- 触发 Kernel Oops：Oops: 0002 [#2] PREEMPT SMP NOPTI

#问题根因
未明确。故障发生在 GPU 挂起后的 TDR（超时检测与恢复）处理过程中，触发了空指针解引用，具体代码缺陷位置尚未确定。

#解决方案
暂无
