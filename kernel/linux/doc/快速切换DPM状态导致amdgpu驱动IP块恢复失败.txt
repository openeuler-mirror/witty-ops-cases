#标题
快速切换DPM状态导致amdgpu驱动IP块恢复失败

#内核版本
5.3.0-050300rc8-generic

#问题现象
- 在RX 540显卡上，快速（间隔小于1秒）切换 `/sys/class/drm/card1/device/power_dpm_force_performance_level` 的值（在low、high、auto之间）时触发。
- 内核日志报错：
  - `[drm:amdgpu_device_ip_resume_phase2 [amdgpu]] *ERROR* resume of IP block <sdma_v3_0> failed -110`
  - `[drm:amdgpu_device_ip_suspend_phase2 [amdgpu]] *ERROR* suspend of IP block <powerplay> failed -22`
  - `[drm:amdgpu_device_ip_resume_phase2 [amdgpu]] *ERROR* resume of IP block <gfx_v8_0> failed -110`

#问题根因
未明确。故障由快速切换电源管理状态触发，导致amdgpu驱动的IP块（如sdma_v3_0、gfx_v8_0、powerplay）在suspend/resume流程中返回错误码（-110 ETIMEDOUT 或 -22 EINVAL）。

#解决方案
暂无
