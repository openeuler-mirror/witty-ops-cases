#标题
USB声卡Behringer Xenyx录音终止时触发NULL指针解引用

#内核版本
4.9.0-8-amd64

#问题现象
- 在终止USB声卡录音操作（调用 snd_pcm_drain 和 snd_pcm_close）时触发内核 Oops
- 内核日志报错：BUG: unable to handle kernel NULL pointer dereference at 0000000000000024
- 崩溃发生在模块 snd_usb_audio 的 set_format 函数中

#问题根因
用户推测可能是应用程序中存在数据竞争，在调用 snd_pcm_drain() 后释放 ring-buffer 导致。内核崩溃点位于 snd_usb_audio 模块的 set_format+0x1d/0x650 处，尝试访问空指针地址。

#解决方案
暂无
