#标题
移除HUGETLB_PAGE_DTOR导致内存压缩时触发kernel BUG

#内核版本
v6.6-rc5

#问题现象
- 运行 XFS fstests 测试（如 xfs/066）时系统触发 kernel BUG。
- 日志显示：`kernel BUG at include/linux/page-flags.h:314!`，伴随 `invalid opcode: 0000 [#1]` 错误。
- 触发断言：`VM_BUG_ON_PAGE`，页面类型显示为 `buddy`。
- 调用栈指向：`folio_flags` -> `PageHuge` -> `isolate_migratepages_block` -> `compact_zone`。

#问题根因
开发者分析认为该问题由 commit 9c5ccf2db04b ("mm: remove HUGETLB_PAGE_DTOR") 引入。在内存压缩流程中，`isolate_migratepages_block` 函数调用 `PageHuge` 检查页面属性时，因页面标志位判断逻辑异常，触发了 `include/linux/page-flags.h:314` 处的 `VM_BUG_ON_PAGE` 断言失败。

#解决方案
暂无修复补丁，开发者建议在相关邮件列表线程中继续讨论。
