#标题
Realtek ALC287 声卡在 Lenovo Legion 7i 上内置扬声器无声

#内核版本
5.11.0

#问题现象
- Lenovo Legion 7i 笔记本电脑内置扬声器无声音输出。
- 连接耳机或外接音箱时声音正常。
- 该问题在 Debian 9、Ubuntu 20.04、Fedora 34、Pop OS 等多个发行版中均存在，Windows 系统下功能正常。

#问题根因
未明确（Bug 报告中未提供具体的代码级根因分析，推测为 Sound(ALSA) 驱动对 Realtek ALC287 在该机型上的适配问题）。

#解决方案
参考 GitHub 仓库 `thiagotei/linux-realtek-alc287` 中 `lenovo-legion` 分支提供的修复方案。
