#标题
Plantronics Savi 700 耳机 HID 接口导致 X Server 锁死

#内核版本
3.9

#问题现象
- Plantronics Savi 700 系列耳机被系统识别为 USB 声卡、键盘和鼠标设备。
- 在 X Window 环境下插入该设备会导致 X Server 锁死，必须拔出设备才能恢复。
- 纯文本控制台下无此问题。
- 使用 evtest 工具检测发现设备发送大量 EV_KEY 和 EV_ABS 事件。

#问题根因
该设备暴露了键盘、鼠标及游戏杆等 HID 接口，并持续发送大量异常输入事件（如 BTN_TASK, BTN_BASE, ABS_MISC 等），导致 X Server 输入子系统处理异常从而引发锁死。

#解决方案
暂无内核补丁修复。可通过在 xorg.conf 中添加配置忽略该设备的输入接口以规避问题：
Section "InputClass"
        Identifier  "Plantronics"
        MatchVendor "Plantronics"
        Option      "Ignore" "true"
EndSection
