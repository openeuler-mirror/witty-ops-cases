#标题
AMD RAVEN GPU modeset触发空指针导致显示黑屏

#内核版本
5.5.6-1-default（问题在 5.5.12 仍存在）

#问题现象
- 系统启动过程中激活 modeset 时屏幕变黑，无法进入图形界面。
- 日志中出现 "BUG: kernel NULL pointer" 报错。
- 系统后台继续引导，可通过远程控制台访问。
- 故障具有偶发性，偶尔（约十分之一概率）能成功初始化并进入 Full HD 图形模式。
- 使用 'nomodeset' 内核参数可启动系统，但分辨率较低。

#问题根因
未明确。日志显示在 modeset 激活瞬间触发了内核空指针错误，导致 GPU 初始化失败。

#解决方案
暂无官方修复。可通过在 GRUB 启动参数中添加 `nomodeset` 规避该问题，但会限制显示分辨率。
