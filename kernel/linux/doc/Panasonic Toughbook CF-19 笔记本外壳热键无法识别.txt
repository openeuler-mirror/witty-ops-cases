#标题
Panasonic Toughbook CF-19 笔记本外壳热键无法识别

#内核版本
3.2.0-33-generic

#问题现象
Panasonic Toughbook CF-19 MK3 笔记本外壳上的 4 个热键（用于屏幕键盘、屏幕旋转等功能）在 Kubuntu 12.04 系统下无响应。
使用 `xev` 和 `acpi_listen` 工具监测时，按下按键均未产生任何事件。
Windows 系统下该设备被识别为 "tablet buttons for panasonic pc"，硬件 ID 为 `ACPI\MAT001F`。

#问题根因
内核未正确识别硬件 ID 为 `ACPI\MAT001F` 的平板按键设备，推测原因为内核缺少对该型号设备热键的驱动支持或 ACPI ID 匹配。

#解决方案
暂无
