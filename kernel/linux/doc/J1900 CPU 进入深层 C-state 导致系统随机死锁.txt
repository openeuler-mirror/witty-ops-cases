#标题
J1900 CPU 进入深层 C-state 导致系统随机死锁

#内核版本
5.10.158-2（用户报告在 6.0.0 版本同样复现）

#问题现象
- 搭载 Celeron J1900 CPU 的系统出现随机冻结，系统无法持续运行超过 24 小时。
- 故障复现于内核版本 5.10 及 6.0.0。

#问题根因
intel_idle 驱动管理电源状态时，允许 CPU 进入深层 C-state，但该特定 CPU（J1900）或其固件在从深层睡眠状态唤醒时存在缺陷，导致系统挂起。该问题类似于历史 Bug 109051。

#解决方案
在内核启动参数中添加 `intel_idle.max_cstate=1`，限制 CPU 仅使用 C1 状态，可规避此问题。目前上游尚未合并正式修复补丁。
