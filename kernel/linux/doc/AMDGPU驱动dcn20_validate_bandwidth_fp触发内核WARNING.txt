#标题
AMDGPU驱动dcn20_validate_bandwidth_fp触发内核WARNING

#内核版本
5.11.4

#问题现象
- 用户在使用 GPU 直通运行 Windows 虚拟机时，主机系统出现短暂卡顿，虚拟机内游戏崩溃。
- dmesg 日志中出现内核 WARNING 堆栈跟踪，位置在 `drivers/gpu/drm/amd/amdgpu/../display/dc/dcn20/dcn20_resource.c:3238`。
- 调用栈显示：`dcn20_validate_bandwidth_fp+0x8d/0xd0 [amdgpu]` -> `dcn20_validate_bandwidth` -> `dc_validate_global_state` -> `amdgpu_dm_atomic_check`。

#问题根因
内核在 AMDGPU 显示核心（DCN 2.0）的带宽验证函数 `dcn20_validate_bandwidth_fp` 中触发了断言失败（WARNING），具体触发的代码逻辑条件未在报告中详细说明。

#解决方案
暂无
