#标题
Radeon RX 6400 显卡因 ASPM 功能异常导致系统花屏死机

#内核版本
5.19-rc3

#问题现象
- 使用 Radeon RX 6400 显卡输出时，屏幕出现花屏，数秒后黑屏
- 系统完全死机，Ctrl-Alt-Del 和 SysRq 键均无响应
- 若使用核显输出且插有 RX 6400，可能导致系统自动重启

#问题根因
amdgpu 驱动在处理 Radeon RX 6400 显卡的 ASPM (Active State Power Management) 电源管理状态时存在缺陷，导致硬件无法正常工作。

#解决方案
在内核启动参数中添加 `amdgpu.aspm=0` 禁用 ASPM 功能，可规避该故障。
