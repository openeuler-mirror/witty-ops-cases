#标题
Lenovo Legion 7 Pro 笔记本扬声器无低音且声音单薄

#内核版本
6.12.22

#问题现象
- Lenovo Legion 7 Pro Gen10 (16IAX10H) 等机型在 Linux 下内置扬声器声音单薄，缺乏低音，低音扬声器未被激活。
- Windows 系统下音频工作正常，问题仅出现在 Linux 环境。
- 尝试加载 snd-soc-aw88399 模块或手动通过 i2c 工具初始化均无法解决问题。

#问题根因
该机型硬件配置发生变更，Smart Amp 芯片从之前的 TAS 系列更换为 Awinic AW88399（ACPI ID 为 "AWDZ8399"）。当前内核 ALSA 驱动对该新型号功放芯片缺乏完善支持，无法正确初始化扬声器配置，导致低音单元无法工作。

#解决方案
暂无官方合并的修复补丁。
社区提供的临时解决方案：参考 GitHub 项目 nadimkobeissi/16iax10h-linux-sound-saga 编译包含特定驱动支持的内核，可恢复扬声器功能。用户反馈该方法有效，但可能存在音量调节逻辑异常或内置麦克风识别问题。
