#标题
NFSv3负载下挂载点挂起导致进程D状态阻塞

#内核版本
4.19.37, 4.19.63

#问题现象
- NFSv3 挂载点在负载下发生挂起
- 内核日志显示 "nfs: server ... not responding, still trying" 以及 "INFO: task ... blocked for more than 120 seconds"
- 进程处于 D 状态（不可中断睡眠），Call Trace 显示阻塞点位于 `rwsem_down_write_failed`（调用链：`grab_super` -> `sget_userns` -> `nfs_fs_mount_common`）或 `nfs_lock_and_join_requests`（调用链：`nfs_updatepage` -> `nfs_write_end`）

未明确（报告仅提供了故障现场堆栈，未给出确定的代码级根因分析）

#解决方案
