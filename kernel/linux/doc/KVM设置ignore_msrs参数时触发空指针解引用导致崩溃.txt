#标题
KVM设置ignore_msrs参数时触发空指针解引用导致崩溃

#内核版本
5.9

#问题现象
当设置 ignore_msrs 参数时，KVM 模块发生崩溃，系统日志显示 kernel NULL pointer dereference。该问题出现在 AMD Ryzen 3700x 硬件平台上。

#问题根因
自 commit 12bc2132b15e0a969b3f455d90a5f215ef239eff 引入。函数 kvm_msr_ignored_check() 在处理 kvm vm get msr features 时，未考虑 vcpu 为 NULL 的情况，直接进行了解引用操作。

#解决方案
开发者计划修复 kvm_msr_ignored_check() 函数逻辑，增加对 vcpu 为 NULL 情况的判断，后续将发布修复补丁。
