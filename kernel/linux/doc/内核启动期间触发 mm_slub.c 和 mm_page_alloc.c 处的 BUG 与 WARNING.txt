#标题
内核启动期间触发 mm/slub.c 和 mm/page_alloc.c 处的 BUG 与 WARNING

#内核版本
6.6.1

#问题现象
- 内核日志报告 `kernel BUG at mm/slub.c:448!`
- 内核日志报告 `WARNING: CPU: 2 PID: 295 at mm/page_alloc.c:4402 __alloc_pages+0x341/0x350`
- 故障发生在 EFI 分区文件系统检查及 MTD 分区创建期间，涉及进程 `udev-worker`。

#问题根因
未明确。日志显示故障发生在 Intel SPI 控制器（0000:00:1f.5）创建 MTD 分区时；用户推测可能与 Dell WMI/BIOS 接口或 EFI 分区 fsck 操作有关。

#解决方案
暂无
