#标题
nftables加载大规则集时报错"File exists"导致加载失败

#内核版本
6.18.15

#问题现象
- 升级至 6.18.15 内核后，nftables 无法加载大型规则集，而在 6.18.13 版本下正常工作。
- 执行加载操作时报错：`Error: Could not process rule: File exists`。
- 若将规则集文件缩短或逐条添加元素则可成功加载。

#问题根因
该问题由 commit `12b1681793e9b7552495290785a3570c539f409d` 引入的回归导致，具体影响 nftables 处理大型规则集的逻辑。

#解决方案
暂无官方修复补丁。临时规避方法为回退问题 commit `12b1681793e9b7552495290785a3570c539f409d`，或采用逐条添加元素的方式加载规则。
