#标题
Radeon驱动运行高负载应用时触发GPU lockup导致系统冻结

#内核版本
4.4, 4.5-rc7, 4.5.1

#问题现象
- 使用 DRI_PRIME=1 运行 Unigine Valley 基准测试时系统完全冻结。
- 内核日志显示 `ring 0 stalled for more than 20040msec`。
- 报告 `GPU lockup` 错误，并伴随 `failed to get a new IB (-35)` 与 `Failed to get ib !` 报错。
- 驱动程序尝试执行 GPU softreset，日志显示 `GPU reset succeeded`，但系统依然无响应。

#问题根因
未明确。日志显示 radeon 驱动检测到 ring 0 停顿超过 20 秒后触发 GPU 锁死，随后获取 IB（Indirect Buffer）失败。尽管驱动尝试重置 GPU 并报告成功，但未能恢复系统正常运行。

#解决方案
暂无
