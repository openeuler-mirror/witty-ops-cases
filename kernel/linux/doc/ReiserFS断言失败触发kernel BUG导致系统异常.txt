#标题
ReiserFS断言失败触发kernel BUG导致系统异常

#内核版本
3.18.21

#问题现象
- 系统触发 kernel BUG，日志显示 "kernel BUG at fs/reiserfs/prints.c:371!"
- 触发 invalid opcode 异常（0000 [#1]）
- ReiserFS 文件系统 panic，报错 "assertion failure check_internal_node: (!(!bh)) ... PAP-12336: bh == 0"
- 故障发生在 rsync 进程调用 SyS_link 系统调用期间

#问题根因
在 ReiserFS 执行文件链接操作时，`do_balance` 流程中调用 `check_internal_node` 检查内部节点，发现 buffer head (`bh`) 为 NULL，触发断言失败（PAP-12336: bh == 0），进而调用 `__reiserfs_panic` 导致系统崩溃。

#解决方案
暂无。建议尝试升级内核版本验证问题是否仍然存在。
