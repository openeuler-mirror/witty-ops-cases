#标题
snd-vx222驱动模块越界访问导致内核崩溃

#内核版本
4.4.0-21-generic（评论中涉及 3.13.0 至 4.8.0 多个版本）

#问题现象
- 使用 Digigram VX222 声卡时，通过 PulseAudio GUI 调整音量或播放音频一段时间后触发内核崩溃
- 系统日志显示 "vx: cannot start stream" 或 "vx: cannot start pipe" 错误信息
- 内核堆栈跟踪显示存在越界访问
- 部分情况下伴随静电噪音或蜂鸣声，随后系统死机

#问题根因
根据内核开发者分析，堆栈跟踪显示 snd-vx222 模块存在越界访问问题。具体触发场景包括音量调节操作或高采样率（如 192kHz）音频流启动。

#解决方案
暂无正式修复。开发者曾提供测试补丁尝试修复越界访问，但测试结果显示问题仍存在。
