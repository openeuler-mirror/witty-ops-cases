#标题
AMD C-60 处理器 Turbo Core 功能在高负载下未激活

#内核版本
3.2.0（问题复现于 4.7.5）

#问题现象
AMD C-60 处理器在 Linux 系统下无法启用 Turbo Core 功能，高负载时 CPU 频率被限制在 1.0GHz，无法达到预期的 1.333GHz。
通过 sysfs 接口查看 `scaling_available_frequencies` 仅显示 1000000 (1GHz) 和 800000 (800MHz)。
使用 turbostat 工具监测显示 Bzy_MHz 始终未超过 TSC_MHz（约 997MHz），确认硬件未进入加速模式。

#问题根因
未明确。内核开发者确认 cpufreq 接口不显示 boost 频率属于正常设计，实际频率需通过读取 APERF/MPERF MSRs 监测。但经用户使用 turbostat 验证，处理器确实未在满载时触发硬件加速，具体驱动层面的故障原因尚在调查中。

#解决方案
暂无。Bug 状态为 ASSIGNED，尚未提供修复补丁。
