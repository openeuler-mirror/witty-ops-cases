#标题
amdgpu驱动调用ida_free释放未分配ID触发内核WARNING

#内核版本
5.5.4

#问题现象
- 系统日志输出内核警告：`ida_free called for id=32782 which is not allocated.`
- 触发 WARNING 的调用栈为：`ida_free+0x11a/0x130` <- `amdgpu_pasid_free_delayed+0x10c/0x2a0` <- `amdgpu_driver_postclose_kms`。
- 故障发生在运行 OpenCL 应用时，伴随 "Failed to create process VM object" 错误。

#问题根因
amdgpu 驱动在 `amdgpu_pasid_free_delayed` 函数中尝试释放一个未被分配的 PASID（id=32782），导致 IDA（ID Allocator）子系统检测到非法释放操作并触发警告。

#解决方案
暂无
