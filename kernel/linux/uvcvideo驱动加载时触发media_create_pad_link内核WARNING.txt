#标题
uvcvideo驱动加载时触发media_create_pad_link内核WARNING

#内核版本
5.9.2（受影响版本包括 5.10.x、5.11.1）

#问题现象
- 系统启动或加载 uvcvideo 模块时触发内核 WARNING。
- 日志显示：`WARNING: CPU: X PID: XXX at drivers/media/mc/mc-entity.c:669 media_create_pad_link+0xXXX/0xXXX`。
- Call Trace 指向 `uvc_mc_register_entities` 和 `uvc_probe.cold`。
- 伴随错误信息：`uvcvideo: Failed to create links for entity 7` 以及 `uvcvideo: Failed to register entities (-22)`。
- 涉及设备为 HP Webcam（USB ID: 0408:5321 或 05c8:0238）。

#问题根因
uvcvideo 驱动在探测特定 HP Webcam 设备时，调用 `uvc_mc_register_entities` 注册媒体实体。在创建 pad link 过程中（针对 entity 7），触发了 `media_create_pad_link` 函数内的 WARNING 检查，导致实体注册失败。

#解决方案
该问题已在内核版本 5.12.4 中修复，升级内核至该版本或更高版本可解决此问题。
