#标题
amdgpu驱动加载失败触发内核缺页异常Oops

#内核版本
4.5

#问题现象
- 加载 amdgpu 驱动后显示器丢失信号。
- 内核日志显示 "amdgpu: ring 0 test failed" 及 "hw_init 5 failed -22"。
- 系统触发 Kernel Oops，日志报错 "BUG: unable to handle kernel paging request at 0000000400000000"，Call Trace 指向 amdgpu 模块内部。

#问题根因
amdgpu 驱动在初始化过程中 ring 0 测试失败（scratch 寄存器值异常），导致硬件初始化错误；在随后的错误处理或清理路径中，代码访问了无效内存地址 0x0000000400000000，触发内核缺页异常。

#解决方案
暂无
