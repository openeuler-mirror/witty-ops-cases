#标题
uvcvideo驱动加载时触发media_create_pad_link内核WARNING

#内核版本
5.10.0-3-amd64 (5.10.12-1)

#问题现象
- 系统加载 uvcvideo 模块识别 HP Webcam 设备时触发内核 WARNING：`WARNING: CPU: 0 PID: 442 at drivers/media/mc/mc-entity.c:669 media_create_pad_link+0x172/0x1a0`
- 内核日志显示 `uvcvideo: Failed to create links for entity 7` 以及 `Failed to register entities (-22)`
- Call Trace 显示调用链为 `uvc_mc_register_entities` -> `uvc_probe`

#问题根因
uvcvideo 驱动在注册媒体实体过程中，调用 `media_create_pad_link` 函数创建 entity 7 的链接时，触发了该函数内部的校验警告（WARN_ON），导致实体注册失败并返回错误码 -22 (EINVAL)。

#解决方案
暂无
