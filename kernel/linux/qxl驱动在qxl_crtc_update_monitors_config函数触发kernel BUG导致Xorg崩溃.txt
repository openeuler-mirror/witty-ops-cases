#标题
qxl驱动在qxl_crtc_update_monitors_config函数触发kernel BUG导致Xorg崩溃

#内核版本
5.12.6

#问题现象
- 系统日志输出 "kernel BUG at drivers/gpu/drm/qxl/qxl_display.c:293!" 并触发 invalid opcode 异常。
- 故障发生在 QEMU 虚拟机环境，进程为 Xorg。
- 触发 kernel panic/oops，调用栈显示经由 drm_atomic_helper_commit_planes 调用链触发。

#问题根因
qxl 显卡驱动在 `qxl_crtc_update_monitors_config` 函数中触发了 `BUG()` 宏（位于 `drivers/gpu/drm/qxl/qxl_display.c:293`），通常表示驱动内部状态断言失败或检测到非法逻辑；具体触发场景涉及 DRM atomic commit 流程中的 planes 提交操作。

#解决方案
暂无（Bug 状态为 NEW，尚未修复）。
