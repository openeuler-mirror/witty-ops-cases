#标题
Panfrost驱动在特定显示器电源状态下触发MMU Page Fault及GPU超时

#内核版本
v5.17（受影响版本包括 v5.3 至 v5.17-rc1）

#问题现象
- 系统日志中出现大量 Panfrost GPU 驱动报错信息。
- 报错包含 `Unhandled Page fault in AS0`，异常类型为 `TRANSLATION_FAULT_LEVEL1`。
- 伴随 `gpu sched timeout` 超时错误。
- 故障与 HDMI 连接及显示器电源状态强相关：
  - HDMI 未连接时，报错出现。
  - HDMI 连接但显示器未通电时，报错出现。
  - HDMI 连接且显示器通电时，报错延迟出现。

#问题根因
开发者推测可能与显示输出状态有关：当无显示输出时，Buffer Objects (BOs) 可能被释放得过于频繁，导致 GPU 访问已释放的内存区域从而触发 Page Fault。

#解决方案
暂无（Bug 状态为 NEW，尚未修复）。
