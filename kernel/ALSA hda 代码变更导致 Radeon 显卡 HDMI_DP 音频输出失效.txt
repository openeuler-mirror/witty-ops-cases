#标题
ALSA hda 代码变更导致 Radeon 显卡 HDMI/DP 音频输出失效

#内核版本
4.6-git

#问题现象
- Radeon R7-265/270X 显卡通过 HDMI 或 DisplayPort 连接显示器时，音频输出无声。
- 执行 `xrandr --set audio on` 后显示器闪烁但音频仍不可用。
- dmesg 中可能出现 DisplayPort link status failed 或 channel eq failed 错误。

#问题根因
Git 二分定位到提交 739ffee97ed550a2899a925ed3f260fa1e8fa955 ("ALSA: hda - Add hdmi chmap verb programming ops to chmap object") 引入了回归，导致 HDMI 音频通道映射处理异常。

#解决方案
Takashi Iwai 在 LKML 上发布了修复补丁，应用该补丁后问题解决。
