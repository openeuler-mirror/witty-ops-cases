#标题
Btrfs挂载特制镜像时btrfs_find_root函数断言失败触发Kernel BUG

#内核版本
5.8.1

#问题现象
- 挂载特制的 btrfs 镜像文件时触发 Kernel BUG
- 系统日志显示 "kernel BUG at fs/btrfs/root-tree.c:81!" 及 "invalid opcode: 0000 [#1]"
- 调用栈指向 btrfs_find_root 函数

#问题根因
在函数 btrfs_find_root() 中，当 search_key->offset 值为 -1ULL 时，代码逻辑进入 else 分支并执行 BUG_ON(ret == 0) 断言。由于特制镜像导致查找结果 ret 返回值为 0，触发了该断言失败，引发 Kernel Panic。

#解决方案
暂无
