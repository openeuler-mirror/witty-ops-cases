#标题
mwifiex_usb 驱动缺失 88W8786U 设备 ID 导致无线网卡无法识别

#内核版本
5.9（评论中也提及 5.4）

#问题现象
- 插入 Marvell 88W8786U USB 无线网卡后，驱动模块 mwifiex_usb 未自动加载
- 手动通过 new_id 接口添加设备 ID (1286:203c) 后，内核日志提示 "mwifiex_usb: unknown id_product 0x203c"
- 设备尝试下载固件 usb8797_uapsta.bin 后立即注销，日志显示 "info: _mwifiex_fw_dpc: unregister device"，网卡无法正常工作

#问题根因
mwifiex_usb 驱动代码中未包含对 USB ID 1286:203c (Marvell 88W8786U) 的定义与支持。驱动程序在检测到未知的产品 ID 后，虽然尝试按默认流程加载固件，但因缺乏针对该型号的特定初始化逻辑，导致设备注册失败并注销。

#解决方案
暂无官方修复。用户尝试修改 drivers/net/wireless/marvell/mwifiex/usb.c 源码添加设备 ID，但仅添加 ID 不足以解决问题，可能需要进一步分析硬件差异并调整驱动逻辑。
