#标题
ath3k驱动因缺失固件文件导致蓝牙加载失败

#内核版本
3.15.4

#问题现象
- 蓝牙功能无法正常使用
- 内核日志报错：Bluetooth: Patch file not found ar3k/AthrBT_0x11020000.dfu
- 内核日志报错：Bluetooth: Loading patch file failed
- 内核日志报错：Direct firmware load failed with error -2
- ath3k 设备探测失败：ath3k: probe of 2-1.6:1.0 failed with error -12

#问题根因
系统缺少蓝牙适配器所需的特定固件文件 `ar3k/AthrBT_0x11020000.dfu`，导致驱动程序无法加载补丁，进而引发 `ath3k` 模块探测失败。

#解决方案
暂无
