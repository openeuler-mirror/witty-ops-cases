#标题
media: mediatek: encoder: Fix uninitialized scalar variable issue

#内核版本
OLK-6.6

#问题现象
内核静态分析工具（UNINIT checker）检测到 Mediatek V4L2 视频编码器驱动中存在变量未初始化即被使用的问题。例如 `enc_result.is_key_frm` 等变量在未初始化状态下被读取，可能导致编码器行为不可预测或功能异常。

#主要错误日志
未知

#问题根因
驱动代码中声明的局部变量（如 `struct venc_enc_param param` 和 `struct venc_done_result enc_result`）未进行初始化。这导致结构体成员（如 `enc_result.is_key_frm`）包含随机的栈数据，后续代码逻辑直接依据这些未初始化的值进行判断或处理，引发了不可预测的行为。

#解决方案
在变量声明时将其初始化为零。具体修改为将 `struct venc_enc_param param;` 修改为 `struct venc_enc_param param = { };`，以及将 `struct venc_done_result enc_result;` 修改为 `struct venc_done_result enc_result = { };`，确保结构体所有成员在使用前均有确定的初始值。
