#标题
amdgpu: VCN v2.0 编解码环不支持 64 位 user fence 导致提交失败

#内核版本
OLK-6.6

#问题现象
在 AMD GPU VCN v2.0 硬件上，当用户态提交包含 user fence 的命令流时，由于硬件不支持 64 位 user fence 写入，导致提交被错误处理或引发功能异常。

#主要错误日志
未知

#问题根因
VCN v2.0 的编码器和解码器环在硬件层面上不支持 64 位 user fence 写入，但驱动程序未设置相应的标志位来拒绝此类 CS 提交，导致尝试执行不支持的操作。

#解决方案
在 VCN v2.0 的解码环和编码环的函数结构体初始化中，将 `no_user_fence` 标志位设置为 `true`，以此拒绝包含 user fence 的 CS 提交，确保硬件操作合规。
