#标题
media: ov534-ov772x 驱动 set_frame_rate 函数存在差一错误

#内核版本
OLK-6.6

#问题现象
在使用 ov534-ov772x 摄像头驱动时，设置帧率（set_frame_rate）功能异常。当用户请求的帧率匹配 rate_0 或 rate_1 数组中的最后一个元素时，驱动无法正确选中该帧率，导致帧率配置与预期不符。

#主要错误日志
未知

#问题根因
在 `set_frame_rate()` 函数中，通过循环遍历数组 `rate_0` 或 `rate_1` 来选择合适的帧率。循环条件 `while (--i > 0)` 存在差一错误，导致循环在索引 `i` 递减至 0 时终止，未能检查数组中的最后一个元素，从而导致边界条件下的帧率选择失败。

#解决方案
修正循环判断条件，将 `while (--i > 0)` 修改为 `while (--i >= 0)`，确保循环能够遍历并检查数组中的最后一个元素（索引为 0 的元素）。
