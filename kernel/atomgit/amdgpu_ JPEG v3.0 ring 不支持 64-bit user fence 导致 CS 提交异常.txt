#标题
amdgpu: JPEG v3.0 ring 不支持 64-bit user fence 导致 CS 提交异常

#内核版本
OLK-6.6

#问题现象
在 Sienna_Cichlid 显卡上使用 JPEG v3.0 引擎时，若用户态提交包含 64-bit user fence 的命令流，会导致硬件无法正确处理该 fence 写入操作。

#主要错误日志
未知

#问题根因
JPEG v3.0 ring 硬件本身不支持 64-bit user fence 写入功能，但驱动程序未设置 `no_user_fence` 标志位，导致内核未能拒绝此类不支持的 CS 提交，进而引发硬件处理异常。

#解决方案
在 `jpeg_v3_0_dec_ring_vm_funcs` 结构体配置中，将 `no_user_fence` 成员设置为 `true`，以此拒绝携带 user fence 的 CS 提交，防止硬件故障。
