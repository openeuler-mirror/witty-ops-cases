#标题
API devm_of_phy_provider_unregister() 无法注销 PHY provider 导致资源泄漏

#内核版本
OLK-6.6

#问题现象
调用 API devm_of_phy_provider_unregister() 后，PHY provider 未被成功注销，导致内存泄漏和 OF 节点引用计数泄漏。

#主要错误日志
未知

#问题根因
API devm_of_phy_provider_unregister() 内部错误地使用了 devres_destroy() 函数。devres_destroy() 不会调用释放回调函数 devm_phy_provider_release()，导致 of_phy_provider_unregister() 未被执行，进而导致 PHY provider 注销失败及相关资源泄漏。

#解决方案
将 devm_of_phy_provider_unregister() 函数中的 devres_destroy() 调用替换为 devres_release()，确保 devm_phy_provider_release() 被正确调用，从而执行注销操作并释放资源。
