#标题
HID: multitouch: Goodix PID 0x01e9 设备 ID 映射错误导致支持缺失

#内核版本
OLK-6.6

#问题现象
Goodix GT7868Q 触摸屏设备（USB PID 为 0x01e9）无法被内核正常识别为多点触控设备，导致触摸功能异常或完全不可用；同时内核代码中存在冗余的 0x01e8 设备条目。

#主要错误日志
未知

#问题根因
在提交 c8000deb68365b ("HID: multitouch: Add support for GT7868Q") 中，开发者意图添加对 Goodix 设备 ID 0x01e8 和 0x01e9 的支持，但由于笔误，`mt_device[]` 数组中添加了两个针对 0x01e8 的重复条目，而遗漏了 0x01e9 的条目，导致 PID 为 0x01e9 的设备无法匹配正确的驱动程序配置。

#解决方案
修改 `drivers/hid/hid-multitouch.c` 文件中的 `mt_devices` 数组，将重复的 `I2C_DEVICE_ID_GOODIX_01E8` 条目修正为 `I2C_DEVICE_ID_GOODIX_01E9`，从而恢复对 Goodix PID 0x01e9 设备的正确支持。
