#标题
arm64: dts: ti: k3-j7200: Fix register map for main domain pmx

#内核版本
OLK-6.6

#问题现象
TI J7200 平台设备树中 main domain 的 pinmux 寄存器映射定义错误，导致 pinctrl 驱动使用的内存映射不正确，可能引发寄存器访问异常、引脚复用功能配置失败或设备（如 USB、MMC）功能异常。

#主要错误日志
未知

#问题根因
之前的修复提交（0d0a0b441346）为了处理不可寻址区域，将 main_pmx0 拆分为 main_pmx0 和 main_pmx1 两个节点，但该提交错误地表示了寄存器范围。这导致 pinctrl 的内存映射与硬件手册（TRM）定义不符。

#解决方案
根据 J7200 TRM (Table 5-695. CTRL_MMR0 Registers) 修正 pinmux 寄存器映射：
1. 修正 main_pmx1 节点，地址调整为 0x11c110，大小为 0x004。
2. 新增 main_pmx2 节点，地址为 0x11c11c，大小为 0x00c。
3. 新增 main_pmx3 节点，地址为 0x11c164，大小为 0x008。
4. 更新板级设备树引用，将 USB0_DRVVBUS 引脚对应的节点引用从 main_pmx1 修改为 main_pmx2，以适配新的地址映射布局。
