#标题
amdgpu: JPEG v2.0 ring 不支持 64-bit user fence 写入导致提交失败

#内核版本
OLK-6.6

#问题现象
在 AMD GPU 设备上使用 JPEG v2.0 环进行计算时，若用户态提交包含 user fence 的命令流，由于硬件不支持 64 位 user fence 写入，会导致命令流提交被拒绝或引发功能异常。

#主要错误日志
未知

#问题根因
JPEG rings 硬件本身不支持 64-bit user fence 写入操作，但驱动程序未在初始化时设置相应的标志位来拒绝此类提交，导致不支持的命令流下发到硬件。

#解决方案
在 `jpeg_v2_0_dec_ring_vm_funcs` 结构体中添加 `.no_user_fence = true` 属性。该设置明确告知驱动层 JPEG v2.0 ring 不支持用户围栏，从而在提交阶段直接拒绝包含 user fence 的 CS submissions，避免硬件层面的异常。
