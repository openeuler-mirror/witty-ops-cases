#标题
cpufreq: acpi-cpufreq 在 Zhaoxin/Centair CPU 上 max-freq 计算错误

#内核版本
OLK-6.6

#问题现象
在 Zhaoxin 或 Centaur CPU 平台上，acpi-cpufreq 驱动计算的 CPU 最大频率（max frequency）不正确，可能导致 CPU 频率显示异常或性能调度策略失效。

#主要错误日志
未知

#问题根因
在 acpi-cpufreq 驱动的 `get_max_boost_ratio` 函数中，未针对 Zhaoxin 和 Centaur CPU 的特性进行特殊处理，导致 max frequency 计算逻辑错误。

#解决方案
在 `get_max_boost_ratio` 函数中增加对 CPU 厂商的判断，若检测到当前 CPU 为 `X86_VENDOR_ZHAOXIN` 或 `X86_VENDOR_CENTAUR`，直接返回 0 作为 workaround，规避错误的频率计算逻辑。
