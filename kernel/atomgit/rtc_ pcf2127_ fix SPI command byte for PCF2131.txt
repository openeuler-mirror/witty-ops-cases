#标题
rtc: pcf2127: fix SPI command byte for PCF2131

#内核版本
OLK-6.6

#问题现象
PCF2131 RTC 芯片在使用 SPI 接口时无法响应读写操作，导致驱动加载失败或 RTC 时间读写功能异常。

#主要错误日志
未知

#问题根因
PCF2131 的 SPI 命令字节定义与 PCF2127/PCF2129 不同，驱动代码在 SPI 模式下未针对 PCF2131 设置正确的命令字节标志位，导致通信失败。

#解决方案
在 SPI 初始化流程中增加针对 PCF2131 型号的判断，当检测到 PCF2131 时，将 `read_flag_mask` 和 `write_flag_mask` 均设置为 0x0，以匹配该芯片的命令字节定义。
