#标题
dt-bindings: net: xlnx,axi-ethernet: Correct phy-mode property value

#内核版本
OLK-6.6

#问题现象
Xilinx AXI Ethernet 设备树绑定文档中 `phy-mode` 属性值 `1000BaseX` 定义错误，与内核驱动及设备树 ABI 标准不符，可能导致设备树解析校验失败或网卡设备无法正常识别与工作。

#主要错误日志
未知

#问题根因
在将 xlnx,axi-ethernet 绑定文档转换为 yaml 格式时（提交 cbb1ca6d5f9a），错误地将 phy-mode 属性值写成了 `1000BaseX`，而内核代码中定义的标准字符串值为 `1000base-x`（全小写）。大小写不一致导致属性值不合法。

#解决方案
修改 Documentation/devicetree/bindings/net/xlnx,axi-ethernet.yaml 文件，将 phy-mode 属性的枚举值从 `1000BaseX` 修正为 `1000base-x`，使其符合内核设备树规范。
