#标题
vcap_api_encode_rule_test 函数内存泄漏

#内核版本
OLK-6.6

#问题现象
在执行 vcap_api_encode_rule_test KUnit 测试用例时，由于测试结束时未释放分配的规则对象，导致内存泄漏。

#主要错误日志
未知

#问题根因
测试函数 vcap_api_encode_rule_test 在完成规则删除操作（vcap_del_rule）后，未调用 vcap_free_rule 函数释放 rule 对象，导致内存泄漏。

#解决方案
在 vcap_api_encode_rule_test 函数末尾、vcap_del_rule 调用之后，增加 vcap_free_rule(rule) 调用以释放内存资源。
