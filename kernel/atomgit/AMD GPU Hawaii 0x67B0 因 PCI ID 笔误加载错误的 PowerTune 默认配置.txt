#标题
AMD GPU Hawaii 0x67B0 因 PCI ID 笔误加载错误的 PowerTune 默认配置

#内核版本
OLK-6.6

#问题现象
AMD Hawaii 系列 GPU 中 PCI ID 为 0x67B0 的设备无法正确加载 `defaults_hawaii_xt` 的 PowerTune 默认参数，导致该特定型号显卡的电源管理功能（如功耗控制、频率调节）行为异常，可能引发性能问题或功耗不符合预期。

#主要错误日志
未知

#问题根因
代码中存在 PCI ID 拼写错误，将 `0x67B0` 误写为 `0x66B0`（`0x66B0` 实际上不存在对应的 AMD GPU）。这导致 0x67B0 设备无法匹配到正确的 case 分支，从而无法加载正确的 `defaults_hawaii_xt` 配置。

#解决方案
修正 drivers/gpu/drm/amd/pm/powerplay/smumgr/ci_smumgr.c 文件中的 PCI ID 判断逻辑，将错误的 `0x66B0` 更正为 `0x67B0`，确保该设备能正确匹配并加载 Hawaii XT 的 PowerTune 默认配置。
