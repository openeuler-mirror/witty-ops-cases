#标题
amdgpu: cyan_skillfish2 GPU info 固件处理逻辑错误

#内核版本
OLK-6.6

#问题现象
在支持 IP discovery（发现二进制块）的 cyan_skillfish2 板卡上，驱动程序未能正确跳过 GPU info 固件的解析流程，导致驱动加载或初始化过程中可能出现功能异常或错误。

#主要错误日志
未知

#问题根因
对于 cyan_skillfish2 这类支持 IP discovery 的板卡，驱动本应直接使用 discovery 表获取信息，无需额外解析 GPU info 固件。但在代码逻辑中，缺少对 `discovery_bin` 是否存在的检查，导致驱动在具备 IP discovery 能力的情况下仍尝试解析 gpu info 固件，引发处理逻辑错误。

#解决方案
在 `amdgpu_device_parse_gpu_info_fw` 函数中针对 `CHIP_CYAN_SKILLFISH` 分支增加判断：若检测到 `adev->mman.discovery_bin` 存在，则直接返回 0，跳过后续的固件解析流程。
