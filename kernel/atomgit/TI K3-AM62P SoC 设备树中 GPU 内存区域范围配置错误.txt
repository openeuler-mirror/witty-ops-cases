#标题
TI K3-AM62P SoC 设备树中 GPU 内存区域范围配置错误

#内核版本
OLK-6.6

#问题现象
TI AM62P5 SoC 平台的 BXS-4-64 GPU 设备树节点中配置的内存区域与硬件手册不符，可能导致 GPU 驱动无法正确初始化、内存访问越界或功能异常。

#主要错误日志
未知

#问题根因
设备树文件 `k3-am62p.dtsi` 中 GPU 的 memory ranges 配置错误。原配置为 `<0x00 0x0fd00000 0x00 0x0fd00000 0x00 0x00020000>`，这与技术参考手册（TRM）中定义的主内存映射不一致。

#解决方案
根据 TI TRM 文档修正 GPU 的内存区域范围。将 `k3-am62p.dtsi` 中 GPU 节点的基地址由 `0x0fd00000` 修改为 `0x0fd80000`，区域大小由 `0x00020000` 修改为 `0x00080000`。
