#标题
AMD R9 280X 显卡运行游戏性能低下并伴随 GPU VM Faults

#内核版本
5.4.0-2-amd64

#问题现象
- 使用 AMD R9 280X (Tahiti) 显卡运行游戏（如 Tomb Raider, Dirt Rally）时性能严重不足，帧率远低于预期。
- 游戏过程中 GPU 使用率异常低（约 18%），但在特定场景（如游戏菜单）下可达到正常水平。
- dmesg 日志中持续输出 `amdgpu 0000:01:00.0: GPU fault detected` 及 `VM_CONTEXT1_PROTECTION_FAULT` 错误信息。
- 性能表现不稳定，部分情况下更改图形设置或重复运行基准测试后性能会暂时恢复正常。

#问题根因
内核日志明确显示 GPU 发生了 VM Context Protection Fault（虚拟内存保护错误）。讨论指出 Southern Islands (Tahiti) 系列硬件可能存在已知的 "mip mapping" 性能下降问题或 Shader 编译/缓存机制异常，导致渲染管线故障及性能波动，但具体代码级根因在报告中尚未最终确认。

#解决方案
暂无。该 Bug 处于 NEW 状态，尚无官方修复补丁。
