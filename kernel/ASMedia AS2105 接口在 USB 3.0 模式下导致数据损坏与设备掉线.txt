#标题
ASMedia AS2105 接口在 USB 3.0 模式下导致数据损坏与设备掉线

#内核版本
3.14.4

#问题现象
- 系统日志被大量 USB 重置和错误信息刷屏，包含 "reset SuperSpeed USB device number 3 using xhci_hcd"、"Parent hub missing LPM exit latency info" 以及 "xHCI xhci_drop_endpoint called with disabled ep" 等报错
- 使用数分钟后，硬盘文件系统发生损坏
- 硬盘设备被卸载或掉线
- 问题仅在 USB 3.0 模式下复现，USB 2.0/1.1 模式下工作正常

#问题根因
ASMedia AS2105 控制器在 USB 3.0 模式下与 xhci_hcd 驱动交互存在异常，日志显示 LPM（链路电源管理）退出延迟信息缺失，且出现端点状态管理错误（xhci_drop_endpoint called with disabled ep），具体代码级根因未明确。

#解决方案
暂无修复方案，维护者建议将问题报告发送至 linux-usb@vger.kernel.org 邮件列表。
