# 标题
pkgcmd “nsenter: failed to execute hostname，环境缺少hostname软件包”

# 基本信息
- 测试用例: oe_test_nsenter
- 系统版本: openEuler-24.03-LTS-SP3\openEuler-24.03-LTS-SP1\openEuler-22.03-LTS-SP4\openEuler-20.03-LTS-SP4(arm、x86_64)
- 是否问题: 否
- 关联issue: 无
- 关联pr: https://atomgit.com/openeuler/mugen/pull/5337

# 问题现象
nsenter -t 30254 -u hostname 执行失败，
