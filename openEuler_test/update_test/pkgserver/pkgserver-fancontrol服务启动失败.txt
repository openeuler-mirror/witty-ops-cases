# 标题
pkgserver “fancontrol服务启动失败”

# 基本信息
- 测试用例: oe_test_service_fancontrol
- 系统版本: openEuler-24.03-LTS-SP3(aarch64/x86_64)
- 是否问题: 否
- 关联issue: 无

# 问题现象
测试执行过程中，检查到fancontrol服务启动失败

# 错误日志
```
+ systemctl start fancontrol.service
+ CHECK_RESULT 0 0 0 'fancontrol.service start failed'
+ actual_result=0

```

# 问题根因
1. 根因在于物理机硬件没有一个专门负责监控温度和控制风扇转速的超级 IO 芯片（比如常见的 ITE IT86xx 系列，或者 Nuvoton NCT67xx 系列）。

# 解决方案
1. 非问题，环境不支持，修改用例，检测环境中是否存在监控温度和控制风扇转速的传感器
修复pr: https://gitcode.com/openeuler/mugen/pull/5333
