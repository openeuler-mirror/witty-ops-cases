# 标题
pkgserver “stap-exporter服务启动失败”

# 基本信息
- 测试用例: oe_test_service_stap-exporter
- 系统版本: openEuler-24.03-LTS-SP3(aarch64/x86_64)
- 是否问题: 否
- 关联issue: 无

# 问题现象
测试执行过程中，检查到stap-exporter服务未启动，且日志报错kernel-debuginfo软件包不存在

# 问题根因
1. 根因在于流流水线基础设施的环境配置缺陷。具体表现为系统未配置 debuginfo对应的 YUM 源，致使测试用例在部署阶段无法定位并安装 kernel-debuginfo包。

# 解决方案
1. 补全软件源配置,添加并启用包含 debuginfo 包的软件仓库地址，确保依赖安装流程正常。
