# 标题
pkgserver “ums_agent服务启动失败”

# 基本信息
- 测试用例: oe_test_service_ums_agent
- 系统版本: openEuler-24.03-LTS-SP3(aarch64/x86_64)
- 是否问题: 否
- 关联issue: 无

# 问题根因
1. 根因在于 ums.ko 作为 umdk-ums 的预编译驱动，与构建时的内核最新版本严格绑定。现网环境内核版本为发布版本的旧内核，与其不兼容，导致模块加载失败。

# 解决方案
1. 更新内核至匹配版本，重启系统后重新加载 ums 模块。
