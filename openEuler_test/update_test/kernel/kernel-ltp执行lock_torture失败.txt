# 标题
kernel "kernel-ltp执行lock_torture失败.txt"

# 基本信息
- 测试用例: lock_torture
- 系统版本: openEuler-24.03-LTS-SP3(aarch64)
- 是否问题：否
- 关联issue：无

# 问题现象
ltp执行过程中报错：failed to load module:locktorture

# 错误日志

'''
failed to load module:locktorture
'''

# 问题根因
1. lock_torture测试依赖内核模块locktorture.ko，但是openeuler内核默认没有编译这个模块，本地执行测试跳过，平台执行报错没有找到模块
解决方案
1. 非问题，如果需要测试，需重新编译内核，启用CONFIG_LOCK_TORTURE_TEST=m
