# 标题
packagement-install-ubs-engine-warning-Group 'ubm_nuds' not found

# 基本信息
- 测试用例: oe_test_pkg_manager01
- 系统版本: openEuler-24.03-LTS-SP3
- 是否问题：否
- 关联issue：无

# 问题现象
pkgmanage测试执行oe_test_pkg_manager01用例过程中，安装软件包usb-engine执行安装脚本时候有提示Warning信息，用例捕捉到warning字段就提示用例执行失败。

# 错误日志

```
[WARN] Group 'ubm_nuds' not found. User 'ubse' was not added to this group. If UBM is required, please install the corresponding package and run: usermod -aG ubm_nuds ubse
```

# 问题根因

ubs-engine软件包在spec中写了安装时判断是否有ubm_nuds这个用户组，如果不存在就抛出告警不打断安装，告知用户缺少 UBM 相关依赖包，用户没加入对应权限组，并给出手动修复命令，后续装好 UBM 包后手动执行加组即可。

# 解决方案
1、确认ubs-engine被成功安装（rpm -qa | grep ubs-engine 或者 yum list installed | grep ubs-engine）
2、成功该告警忽视
