# 标题
packagement-remove-ubturbo-ubdma-卸载脚本报错modprobe FATAL Module ub_dma not found

# 基本信息
- 测试用例: oe_test_pkg_manager01
- 系统版本: openEuler-24.03-LTS-SP3
- 是否问题：是
- 关联issue：https://gitcode.com/src-openeuler/ubturbo/issues/5

# 问题现象
pkgmanage测试执行oe_test_pkg_manager01用例过程中，卸载软件包ubturbo-ubdma时执行到卸载脚本出现FATAL字样，用例判定失败

# 错误日志

```
modprobe: FATAL: Module ub_dma not found.
error: %preun(ubturbo-ubdma-1.0.0-2.oe2403sp3.aarch64) scriptlet failed, exit status 1

Error in PREUN scriptlet in rpm package ubturbo-ubdma

```

# 问题根因
%preun 里直接执行 modprobe ub_dma 尝试加载模块，正确卸载逻辑应该是卸载已加载模块（rmmod），而不是加载；
同时未做存在性判断


# 解决方案
ub_dma.ko不存在，所以直接spec移除了 %preun ubdma该部分的代码
pr：https://atomgit.com/src-openeuler/ubturbo/pull/48
