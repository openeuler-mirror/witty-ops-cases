# 周报模板（weekly_summary.md）

> 仅周报模式使用。所有流水线的全部模块报告完成后，按此结构在周根目录生成 `weekly_summary.md`。`<周标识>` 形如 `2026-W27`。

```markdown
# openEuler Update 测试周报 - <年份>年第<周序>周

## 总览

| 指标 | 数值 |
|---|---|
| 分析流水线数 | N/N（全部完成） |
| 总失败用例 | N |
| ✅ 自动拦截 | x (x%) |
| ⚠️ 待确认 | x (x%) |
| ❌ 未拦截 | x (x%) |
| 新增案例草稿 | x条已入库 + y条报告内待审核 |

> 拦截率趋势: 上次 x% → 本次 y%（↑/↓ Δ）。<一句话原因说明：本周失败模式构成与上周差异>。

> ⚠️ 数据说明：<若某流水线某模块报告有已知问题，在此注明并给出修正后估计值>。
```
