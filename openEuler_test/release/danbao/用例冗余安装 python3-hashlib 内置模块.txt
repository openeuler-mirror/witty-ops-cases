# 标题
用例冗余安装 python3-hashlib 内置模块

# 基本信息
- 测试用例: oe_test_python3-hashlib
- 系统版本: openEuler-24.03-LTS-SP4
- 是否问题: 否
- 关联issue: 否

# 问题现象
测试前置环境准备阶段执行安装 python3-hashlib，系统源无该安装包，提示包不存在，前置步骤执行失败，直接跳过后续测试流程进入环境清理。

# 问题根因
1. 测试用例预置了安装 python3-hashlib 的指令，但系统软件源中不存在该 rpm 包；hashlib 属于 Python 内置标准模块，无需单独安装，多余的安装命令导致前置环境部署失败。

# 解决方案
1. 删除测试用例中安装 python3-hashlib 的冗余命令
pr：https://gitcode.com/openeuler/mugen/pull/5201
