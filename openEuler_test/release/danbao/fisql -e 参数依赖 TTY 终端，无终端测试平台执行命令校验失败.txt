# 标题
fisql -e 参数依赖 TTY 终端，无终端测试平台执行命令校验失败

# 基本信息
- 测试用例: oe_test_freetds_fisql
- 系统版本: openEuler-24.03-LTS-SP4
- 是否问题: 否
- 关联issue: 无

# 问题现象
执行oe_test_freetds_fisql用例的时候，fisql -e只能在真实终端（TTY）里运行，在非交互、无终端的平台无法执行 fisql -e 命令，因此返回退出码 1 导致用例失败。

# 问题根因
1. fisql -e 执行逻辑强依赖真实 TTY 交互终端，平台为后台无终端非交互环境，缺少标准 TTY 设备；执行带 -e 参数的 fisql 命令输出文件无法生成预期内容，grep 匹配不到目标 SQL 语句，校验返回码 1，用例判定失败。


# 解决方案
1. 非问题，更新测试运行环境，提供真实 TTY 终端环境，保证 fisql -e 命令正常执行。
