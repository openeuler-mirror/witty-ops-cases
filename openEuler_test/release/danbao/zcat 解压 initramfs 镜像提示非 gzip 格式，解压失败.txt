# 标题
zcat 解压 initramfs 镜像提示非 gzip 格式，解压失败

# 基本信息
- 测试用例: oe_test_squashfs-tools
- 系统版本: openEuler-24.03-LTS-SP4（x86_64）
- 是否问题: 否
- 关联issue: 无

# 问题现象
执行oe_test_squashfs-tools用例的时候，当 zcat+cpio 解压 initramfs 镜像时，日志提示文件不是 gzip 格式，cpio 归档提前终止，初始化镜像解压步骤校验失败，用例报错退出。

# 问题根因
1. 测试脚本使用zcat解压 initramfs 镜像，但平台x86 架构的 initramfs 为 SquashFS 格式，并非 zcat 支持的 gzip 格式，工具与镜像格式不匹配，解压抛出报错；

# 解决方案
1. 非问题，更换正常识别 initramfs 镜像压缩格式的测试环境后重新执行用例。
