# 标题
script测试环境中无tty返回退出码 1 导致用例失败

# 基本信息
- 测试用例: oe_test_script
- 系统版本: openEuler-24.03-LTS-SP4
- 是否问题: 否
- 关联issue: 无

# 问题现象
执行oe_test_script用例的时候，script 命令依赖交互终端（TTY），在无 TTY 的非交互自动化环境下无法正常执行，因此返回退出码 1 导致用例失败

# 问题根因
1. 执行环境为无 TTY 终端环境，而script工具纯输出录制模式强依赖外部交互式 TTY 伪终端，所以导致script命令执行后异常返回非 0 码，用例步骤校验失败，生成 ERROR 错误日志。

# 解决方案
1. 非问题，更换可分配交互式 TTY 终端的测试执行环境，重新执行测试
