# 标题
pkgcmd-python-gitlab-访问gitlab私人令牌失效

# 基本信息
- 测试用例: oe_test_gitlab_01 oe_test_gitlab_02
- 系统版本: openEuler-24.03-LTS-SPX
- 是否问题：是
- 关联issue：无

# 问题现象
跑单包用例时候出现bash mugen.sh -f python-python-gitlab -x失败，报：401: invalid_token

# 错误日志

```
oe_test_gitlab_01：
+ gitlab -c tmp.cfg user list
+ grep id:
401: invalid_token
+ CHECK_RESULT 1 0 0 'Check gitlab -c CONFIG_FILE failed'


oe_test_gitlab_02：
+ gitlab -o json -c tmp.cfg user list
+ grep -e '\[{.*}\]'
401: invalid_token
+ CHECK_RESULT 1 0 0 'Check gitlab -o {json,legacy,yaml} failed'


```

# 问题根因
gitlab的tocken失效导致

# 解决方案
重新注册gitlab测试账户，获取tocken
pr：https://atomgit.com/openeuler/mugen/pull/5268
